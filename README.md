# Cardiovascular
## Introduction
This is the machine learning scripts for analysis of ASCVD, CHF and hypertension.

## Installation
```
mamba install -c conda-forge \
  r-pacman r-xgboost r-randomforest r-gbm r-caret r-proc \
  r-ggplot2 r-dplyr r-tidyverse r-gridextra r-rms r-rmda \
  r-pheatmap r-shapviz r-reshape2 r-e1071 r-nnet r-lightgbm \
  r-ada r-glmnet r-catboost
```

### Repository
#### ASCVD:
```
Rscript ML_final2_ASCVD.R
```
#### CHF:
```
Rscript ML_final2_CHF.R
```
#### Hypertension:
```
Rscript ML_final2_hypertension.R
```
